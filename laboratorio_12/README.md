# Laboratorio 12

## Requisitos

Tener instalado:

* Python 3
* NumPy
* Matplotlib
* Pillow

Instalar las dependencias necesarias:

```bash
pip install numpy matplotlib pillow
```

## Ejecución

Desde la carpeta del laboratorio ejecutar:

```bash
python heat_2d.py
```

o bien:

```bash
py heat_2d.py
```

## Salida

Al finalizar la ejecución el programa genera:

* `calor_2d.gif`

Además imprime en pantalla información sobre:

* Paso temporal (`dt`)
* Número de pasos temporales (`num_steps`)
* Cantidad de snapshots almacenados

La animación muestra la evolución temporal de la temperatura en una placa bidimensional con condiciones de frontera constantes.
